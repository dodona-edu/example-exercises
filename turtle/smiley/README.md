A creative, open-ended Turtle exercise where students draw a smiley face using Python's `turtle` module.

Students are encouraged to experiment with colors, shapes, and accessories (eyebrows, nose, rosy cheeks, hats, etc.).
There is no single correct answer — the exercise builds familiarity with turtle graphics and rewards creativity.
Automated grading only checks for syntax errors; the drawing itself is not compared against a reference solution.
